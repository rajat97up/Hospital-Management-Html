What is Android?





Today, there will hardly be anyone who does not know about Android, especially people associated with technology, everyone has used Android, whether it is Android phone or Android Watch or any other Android-based devices, in some form or the other.

So let's know what is Android , who made it and what are the features of Android.

If I tell in simple language, Then Android is a mobile operating system. From the developer's point of view, Android is a software stock.
Android is the fastest growing mobile operating system. You can say that Android has brought R evolution in the mobile operating system. After the arrival of Android, almost all mobile companies have started making Android-based phones, basically these phones are called smartphones, which are really smart, inside which many smart fitters have been given.


The co-founder of A Ndroid is Andy Rubin, who and some of his friends discovered it together, later it was developed by Google and other companies under the Open Handset Alliance, since then it became world wide. Android Linux is a kernal-based operating system. Android is an open source operating system that anyone can customize and make it according to them, the company used to give its operating system with the earlier phones, every company had its own operating system, such as Symbian in N Okia, Bada in Samsung, etc. It is very good that we do not even need to install different apps, the same Android app works in almost all Android devices. And the app developer also does not need to create different apps for different phones or devices, they develop an Android-based app and they run in Android based devices.

Today many versions of Android have come, basically it is also called API level, the new version of Android has now got a lot of features and its user interface has become very improwed. Below are some android versions and their names.

Features of Android

If seen, there are many features of Android, some of which I am going to tell you which I like very much.


User Interface - Android's user interface makes our work very easy, such as in Android smartphones you do not need to find apps, it is found on your main screen so that you can use it if needed.

 Multi touch - Multi touch means that you can operate your smartphone with both fingers, such as if you have to zoom in or zoom out of a photo or any place, then you can do it by touching the screen with both of your own.


Connectivity – Talking about connectivity, you get Wi-Fi, Bluetooth, Gsm / Edge, Cdma, 3G / 4G, Volte, Nfc etc.
